دعوة زفاف إيهاب ودنيا — جاهزة لـ GitHub Pages

الرفع من صفحة GitHub التي وصلت إليها:
1. اضغط uploading an existing file.
2. فك ضغط الملف على الآيفون أولًا.
3. ارفع index.html وstyles.css وscript.js.
4. ارفع مجلد assets وبداخله الصورة والموسيقى.
5. اضغط Commit changes.
6. من Settings > Pages:
   Source: Deploy from a branch
   Branch: main
   Folder: /root
7. Save وانتظر من دقيقة إلى 5 دقائق.

العداد مضبوط على الأحد 6 سبتمبر 2026 الساعة 8 مساءً بتوقيت مصر.
الموسيقى تعمل بعد الضغط على «افتح الدعوة» بسبب قيود المتصفحات.
زر تأكيد الحضور يجهز رسالة واتساب، ثم يختار الضيف محادثة إيهاب أو دنيا.
